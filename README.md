# server-github-edomingorena14
**Parte practica Fentcami Bloque 3, Servidor en la Nube con GitHub**

Este proyecto consiste en la creación y configuración de un servidor Linux en la nube utilizando una máquina virtual integrada en GitHub (Codespaces).
El servidor ofrece:

·Una web pública accesible sin autenticación.

·Una web privada protegida con autenticación básica (usuario y contraseña).

·Acceso remoto mediante SSH.

·Gestión del proyecto y control de versiones con GitHub.

El objetivo principal es entender cómo desplegar servicios básicos en un servidor Linux.

**Explicacion VM integrada en Codespaces**

El proyecto utiliza GitHub Codespaces, que proporciona una máquina virtual Linux basada en Ubuntu directamente vinculada al repositorio.

Características principales del Codespace:

·Sistema operativo: Ubuntu Linux

·Acceso a terminal completo

·Posibilidad de instalar servicios (Apache, SSH…)

·Reenvío de puertos para acceder al servidor web

·Integración directa con Git y GitHub

·Entorno persistente vinculado al repositorio

Esta máquina virtual permite trabajar sin necesidad de instalar nada en el PC local y facilita la configuración del servidor dentro del propio ecosistema GitHub
